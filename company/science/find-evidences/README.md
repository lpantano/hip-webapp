# Paper Finder Agent Results

This directory contains markdown files generated by the Paper Finder Agent.

## What's Here

- **Summary files**: Overview of paper search results for each claim
- **Individual paper files**: Detailed analysis of each found paper
- **Expert review notes**: Space for manual review and notes

## File Naming Convention

- `<claim-title>-summary-<date>.md` - Search summary
- `<claim-title>-paper-<rank>-pmid-<pmid>.md` - Individual paper details

## Usage

1. Run the Paper Finder Agent for a claim
2. Review the generated markdown files
3. Use the interactive review to add approved papers to the database
4. Keep these files as a record of automated searches

## Documentation

See [docs/PAPER_FINDER_AGENT.md](../../../docs/PAPER_FINDER_AGENT.md) for complete documentation.

## Note

These files are for review purposes and are not automatically added to the database.
Each paper must be reviewed and approved before being added to Evidence Decoded.
